## 🚀 Getting Started

### ✅ Prerequisites

Ensure you have these installed:

**Backend Requirements:**
```bash
PHP 8.3+  •  MySQL 8.0+  •  Composer  •  Git  •  Web Server (Apache/Nginx)
```

**Mobile Requirements:**
```bash
Node.js 16+  •  npm or yarn  •  React Native CLI  •  Android Studio / Xcode
```

**Verify versions:**
```bash
php --version           # Should be PHP 8.3+
mysql --version         # Should be MySQL 8.0+
composer --version      # Latest version
node --version          # Should be Node 16+
npm --version           # Latest version
```

### 🔧 Installation & Setup

#### Step 1️⃣ Clone Repository
```bash
git clone https://github.com/Moparapairayat/NoorFi.git
cd NoorFi
```

#### Step 2️⃣ Backend Setup

**Install dependencies:**
```bash
cd backend
composer install
```

**Environment configuration:**
```bash
# Create environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Configure database in .env
# DB_HOST=localhost
# DB_DATABASE=noorfi
# DB_USERNAME=root
# DB_PASSWORD=
```

**Database setup:**
```bash
# Run migrations
php artisan migrate

# Seed initial data (optional)
php artisan db:seed
```

#### Step 3️⃣ Mobile Setup (Optional)

```bash
cd ../mobile

# Install dependencies
npm install
# or
yarn install

# Configure API endpoint in environment
# Update services/api.ts with your backend URL
```

```bash
# For Android
npm run android

# For iOS
npm run ios
```

### ▶️ Running the Project

**Start backend development server:**
```bash
cd backend
php artisan serve
```

**Output:**
```
Laravel development server started: http://127.0.0.1:8000
```

**Start mobile development:**
```bash
cd mobile
npm start
```

🎉 **Open** [http://localhost:8000](http://localhost:8000) in your browser!

## 🎨 Customization Guide

### 🎯 Configuration

Edit `backend/.env`:
```env
APP_NAME=NoorFi
APP_URL=http://localhost:8000
DB_DATABASE=noorfi
DB_USERNAME=root
```

**Mobile API Configuration:**
Edit `mobile/src/services/api.ts`:
```typescript
const API_BASE_URL = 'http://localhost:8000/api';
```

### 📝 Content Management

Manage data through:
- Filament Admin Dashboard at `/admin`
- Direct database modification
- Artisan CLI commands
- Migration files

### 🏢 Add New Features

**Create a new migration:**
```bash
php artisan make:migration create_new_table --create=new_table
```

**Create a new model:**
```bash
php artisan make:model NewModel -m
```

**Create a new controller:**
```bash
php artisan make:controller Api/NewController
```

**Create a Filament resource (Admin):**
```bash
php artisan make:filament-resource NewResource
```

## 🚀 Deployment

### 🔥 Deploy Backend to Web Server

**Using XAMPP (Development):**
1. Place `backend/` folder in `htdocs/`
2. Start Apache & MySQL
3. Access via `http://localhost/backend/public`

**Using Linux Server:**
```bash
# SSH into server
ssh user@your-server.com

# Clone repository
git clone https://github.com/Moparapairayat/NoorFi.git

# Setup backend
cd NoorFi/backend
composer install
cp .env.example .env
php artisan key:generate

# Configure database
# Edit .env with your database credentials

php artisan migrate
php artisan config:cache
php artisan route:cache
```

### 📱 Deploy Mobile App

**For Android:**
```bash
cd mobile
npm run android -- --release
# Follow the build instructions to sign and upload to Google Play Store
```

**For iOS:**
```bash
cd mobile
npm run ios -- --configuration Release
# Follow the build instructions to upload to App Store
```
